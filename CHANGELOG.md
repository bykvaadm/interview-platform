# Changelog

Все заметные изменения проекта. Формат — [Keep a Changelog](https://keepachangelog.com/ru/),
версии по [SemVer](https://semver.org/lang/ru/). Номер версии и эта история зашиваются в страницу при сборке (`build.py` читает верхнюю версию отсюда).

## [0.2.2] - 2026-06-19
### Исправлено
- Установка сессии по WebRTC: устранена ошибка `setRemoteDescription … wrong state: stable`. Защита от повторного применения код-ответа (двойной клик / повторный сигнал), такая ошибка больше не рвёт устанавливающееся соединение. Убран внутренний watchdog-таймер, который мог по ложному срабатыванию сам разрывать живую сессию (keep-alive ping оставлен).

## [0.2.1] - 2026-06-19
### Изменено
- Блок «Данные» (экспорт/импорт конфигурации одним JSON и сброс к исходному) перенесён из вкладки «Матрицы» в «Подготовку» — всё управление данными в одном месте. Конфигурация — единый JSON: вопросы + категории + матрицы (баллов и ожиданий) + пресеты.

## [0.2.0] - 2026-06-19
### Добавлено
- Совместная работа в реальном времени через WebRTC (P2P, без сервера): двусторонняя синхронизация сессии между двумя интервьюерами, с сохранением фокуса/каретки при правках.
- Текстовый чат интервьюеров по тому же каналу: звук входящего сообщения и кнопка 🔔/🔕, группировка подряд идущих сообщений одного автора, сворачивание и закрытие панели.
- Локальное поле «Ваше имя» (у каждого участника своё, не синхронизируется) для подписи в чате; обмен именами при подключении; поле кандидата переименовано в «Интервьюер(ы)» (можно вписать двоих).
- Keep-alive (heartbeat) против обрывов по простою и watchdog обрыва; реконнект в один клик с запоминанием роли и сохранением сессии и чата.
- Опция «Свой TURN» (сохраняется в браузере) для связи между разными сетями.
- Версионирование: номер версии в шапке и история изменений во вкладке «Инструкция».
### Изменено
- CDN библиотеки P2P переключён с unpkg на jsdelivr (надёжнее в ряде регионов, в т.ч. РФ).
- Шапка: на десктопе всегда один ряд (вкладки скроллятся при нехватке места), на мобиле — сворачиваемое бургер-меню; убран подзаголовок.
- В таблице грейдов пояснён столбец «Δ к порогу» (легенда и подсказка).
### Исправлено
- Крестик и сворачивание чата (атрибут `hidden` перебивался `display:flex/grid`).
- Тост дольше висит для длинных сообщений и закрывается по клику; ховер кнопок больше не сливается с фоном.

## [0.1.0] - 2026-06-17
### Добавлено
- Перенос Google-таблицы для собеседований в статическое веб-приложение (GitHub Pages), один файл `index.html`.
- Банк вопросов DevOps/SRE; категории по группам CORE / MEDIUM / SPECIFIC.
- Вкладка «Подготовка»: фильтры по группе/категории/сложности, поиск, набор вопросов в сессию; готовые наборы (пресеты), включая «Базовый набор (как в таблице)» — 29 вопросов из исходного листа.
- Вкладка «Собеседование»: ключевые точки с галочками, оценка рекрутёра, корр. баллы, расчёт суммы баллов и грейда, визуальная шкала порогов. Логика 1:1 с исходной таблицей (10 Low / 15 Medium / 4 High → 12.5 / 44 / 59 / 67 / 75).
- Комментарии по каждому вопросу и общий комментарий по кандидату; сворачивание карточек вопросов.
- Полностью редактируемые банк вопросов и матрицы (баллов и ожиданий); сохранение в localStorage; сброс к исходному; импорт/экспорт конфигурации в JSON.
- Экспорт результатов в Excel (.xlsx) и PDF.
- Светлая/тёмная темы с переключателем; адаптивная вёрстка.
- Источник данных в человекочитаемом YAML (`source/questions.yaml`, `source/presets.yaml`), сборка через `source/build.py`; автотесты (node + jsdom).
